# ECOMMERCE - Backend

### Dependencies

- NodeJS
- NPM

### Setup

- Copy the  `.env.example` to a file named `.env`
- Open the `.env` and replace all missing values

```bash
npm i
npm run generate:db
```

### Development

```bash
npm run dev
```

### Routes

```bash
/health-check
/sign-in
/products
/orders
```
