ECOMMERCE - Back end

A simple API to return that has authentication, products list, and register orders.

/health-check [GET]

Returns a response only to notify that the server is up

Usage

/health-check

/products [GET]

Returns a paginated products list

Query Params

page [number] [default=1]
limit [number] [default=10]

Usage

/products?page=1&limit=10

/sign-in [POST]

Returns the user according the given credentials

Body Payload

Object < username [string], password [string] >

Errors

[401] AUTHENTICATION_FAILED

/orders [GET]

Returns all orders of a user

Headers

Authentication [pattern "Bearer: < token >"]

/orders [POST]

Save a order for a user

Headers

Authentication [pattern "Bearer: < token >"]

Body Payload

items [Array< Object< name [string], price [number], quantity [number] > >]
